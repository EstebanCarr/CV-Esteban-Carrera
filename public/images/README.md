# Imágenes para CV

## Estructura de carpetas:

### `/profile/`
- **profile-photo.jpg** - Tu foto profesional (recomendado: 400x400px)
- **background.jpg** - Imagen de fondo opcional

### `/projects/`
- **eccsa-preview.jpg** - Captura de pantalla de eccsa-web.vercel.app
- **arenaton-preview.jpg** - Captura de pantalla de www.arenaton.com
- **warpcast-preview.jpg** - Captura del proyecto Warpcast Frames

### `/icons/`
- **company-logos/** - Logos de empresas donde has trabajado
- **tech-icons/** - Iconos de tecnologías que manejas

## Formatos recomendados:
- **Fotos**: JPG/PNG
- **Capturas**: JPG/PNG (máximo 1920x1080)
- **Iconos**: SVG/PNG (64x64px o 128x128px)

## Uso en el código:
Las imágenes se referencian como: `/images/carpeta/archivo.jpg`
